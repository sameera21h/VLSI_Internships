# 🚦 Traffic Light Controller using Verilog HDL

## Overview

This project implements a Traffic Light Controller using Verilog HDL based on a Finite State Machine (FSM).

The controller cycles through three traffic light states:

* 🔴 RED
* 🟢 GREEN
* 🟡 YELLOW

The design was verified using QuestaSim.

* traffic_light.v

* run.do

## Tools Used

* Verilog HDL

## FSM Sequence

RED → GREEN → YELLOW → RED
